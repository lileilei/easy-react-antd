```bash
安装全局命令
$ npm install -g  或者 运行 install.bat
new 一个新项目，名称为benny
$ coder --cli benny
在当前目录生成模块名为name1和name2的模块
$ coder name1 name2
在src/routes 目录下新建项目模块地图文档例如 project.js
$ coder --file project.js
```
